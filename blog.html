What is Document Object Model(DOM)?

Answer:The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

What event delegation in the context of the DOM andwhy is it useful?

Answer: Event delegation is basically a pattern to handle events efficiently. Instead of adding an eventListener to each and every similar element. we add an eventListener to a parent element and call an event on a particular target using the .target property of the event object.

How do you manipulate a element's attributes and styles using the DOM?

Answer: Attributes of an element can be manipulated by using methods like getAttribute(), setAttribute(), and removeAttribute() and styles of an element can be manipulated by using the style property or by modifying the classList for applying external styles.
